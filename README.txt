Michango App (PWA) - Green theme
Files included:
- index.html
- style.css
- app.js
- manifest.json
- service-worker.js

How to use:
1. Extract this folder on your computer or phone.
2. Open index.html in a browser (Chrome or Edge recommended).
3. On Android/Chrome: open the browser menu and choose "Add to Home screen" to install PWA.
4. Optionally add icons named icon-192.png and icon-512.png to the folder.

Note: Data is stored locally in the browser (LocalStorage). Export using the Export CSV button.
